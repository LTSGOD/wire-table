# create
데이터를 입력할 수 있도록 `생성`버튼이 활성화 되어 있습니다.


## 버튼 상태제어
Actions값을 통하여 버튼의 기능을 비활성/활성을 제어할 수 있습니다.

### create 비활성화
`생성` 버튼을 비활성화 합니다.
```php
$this->createDisable()
```
> 이 기능은 `BaseController`에 선언되어 있습니다.

### create 활성화
`생성`버튼을 활성화 합니다. 메서드의 인자값으로 출력해야 하는 텍스트를 같이 변경할 수 있습니다.
> 기본값은 `추가`입니다.

```php
$this->createEnable("추가");
```
